# 🏋️ Workout Tracker

A personal fitness tracking application that will grow from a simple workout logger into a full-stack fitness management system.

This project serves two purposes:

- Build a tool that I will actually use every day.
- Learn modern software development by continuously improving a real application.

Instead of creating dozens of disconnected tutorial projects, this repository will evolve alongside my programming skills.

---

# Vision

The goal is to create a fitness tracker that manages every aspect of training:

- Strength Workouts
- Running
- Swimming
- Rucking
- Body Measurements
- Recovery
- Nutrition
- Personal Records
- Progress Analytics
- Workout Planning
- PJ Fitness Test Tracking

By the end of development, this project should demonstrate full-stack software engineering skills while also becoming the primary application I use for my own training.
